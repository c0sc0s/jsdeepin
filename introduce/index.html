欢迎 #

欢迎大家来到 JavaScript Deep In

随着Web的兴起，Js成为了如今编程世界里，十分重要的一门语言。

Js 有着可爱的外表，宽松的语法，大量的Api，似乎是一门很好欺负很好拿捏的小可爱。但当你走入 Js 的世界，你就会发现 Js 可爱的外表下，隐藏了太多的秘密和细节，这些细节会让 Js 产生很多令人抓狂的奇怪现象。

哈哈忽然想起了这个经典老图，在这个就是JS的隐式类型转换，也许你记住了一些规则，那么他底层运行的逻辑是什么！你是否从这个角度出发过！

再者你是否听说过闭包和变量提升？如果你了解，你是否想过这些现象是靠哪些底层的逻辑推导而来的。如果你没听说过，那也完全没有关系，经过我们的这次旅行，他们对你而言不过是水到渠成的东西。

原型链，事件循环，异步 .... Js 有太多太多值得挖掘和思考的内容，当然也有很多人对 Js 的这些内容做了解释，但大多数普遍从现象出发，更多强调他们是什么？而忽略了为什么。这也导致了很多人对这些知识似乎明白但总感觉不是很通透。

本篇教程更多的不是阐述现象本身，因为这太表面，而是尽可能的从这些特征的 需求 出发，用底层的逻辑推导出这些现象。这是一次逻辑的冒险，没有那么多绕弯的名词，我们一同驾驶着小舟在逻辑的海湾冲浪，一点一滴扩大自己对 Js 世界认知的版图。
